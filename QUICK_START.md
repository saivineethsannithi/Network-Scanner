# ⚡ Quick Reference — Network Scanner GitHub

## 🎯 5-Minute Quick Start

### Step 1: Create Repository
```
Go to https://github.com/new
Name: network-scanner
Description: Professional network scanner for device discovery
Visibility: Public
Create repository
```

### Step 2: Get HTTPS Link
```
Click "Code" button
Copy HTTPS URL (looks like: https://github.com/YOUR_USERNAME/network-scanner.git)
```

### Step 3: Upload via Git Terminal

```bash
# Clone your new empty repo
git clone https://github.com/YOUR_USERNAME/network-scanner.git
cd network-scanner

# Copy all files from this package into the folder
# Files should include:
# - network_scanner.py
# - network_scanner_dashboard.html
# - README.md
# - LICENSE
# - requirements.txt
# - .gitignore
# - All other files

# Upload to GitHub
git add .
git commit -m "Initial commit: Network Scanner v2.0.0"
git push origin main
```

### Step 4: Verify
- Go to your GitHub repo page
- Refresh browser
- See README display
- All files listed
- Done! ✅
